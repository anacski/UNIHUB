# UniHub - Projeto Integrador

### Descrição
O **UniHub** é uma plataforma web desenvolvida como Projeto Integrador universitário, feita para otimizar a busca e a organização de atividades acadêmicas por parte dos estudantes. 

O sistema visa conectar alunos a eventos acadêmicos, tais como:
* Atividades físicas, esportivas e de integração.
* Grupos de estudo e monitorias.
* Palestras, workshops e simpósios.

### Objetivos do Projeto
* **Centralização:** Reunir eventos e atividades em um único ecossistema acessível.
* **Acessibilidade:** Oferecer uma interface intuitiva para que os alunos encontrem grupos de interesse comum.
* **Organização:** Auxiliar no planejamento e na rotina extracurricular dos estudantes.
