# UniHub - Projeto Integrador

## Descrição
O **UniHub** é uma plataforma web desenvolvida como Projeto Integrador universitário, projetada para centralizar, facilitar e otimizar a busca e a organização de atividades acadêmicas por parte dos estudantes. 

O sistema visa engajar a comunidade universitária ao conectar alunos a eventos acadêmicos, tais como:
* Atividades físicas, esportivas e de integração.
* Grupos de estudo e monitorias.
* Palestras, workshops e simpósios.

## Objetivos do Projeto
* **Centralização:** Reunir eventos e atividades em um único ecossistema acessível.
* **Acessibilidade:** Oferecer uma interface intuitiva para que os alunos encontrem grupos de interesse comum.
* **Organização:** Auxiliar no planejamento e na rotina extracurricular dos estudantes.

## Estrutura de Tecnologias
O front-end da aplicação foi construído de forma nativa para garantir desempenho e portabilidade:
* **HTML5:** Estruturação semântica das páginas e formulários de busca.
* **CSS3:** Identidade visual e layout responsivo adaptável a dispositivos móveis e desktops.
* **JavaScript (ES6+):** Lógica para filtragem de atividades, interatividade e manipulação dinâmica dos elementos em tela.

## Arquitetura de Arquivos
* `/css`: Folhas de estilo responsáveis pela padronização visual da plataforma.
* `/js`: Scripts para o comportamento dinâmico e regras de negócio no front-end.
* `/imagens`: Repositório de ativos visuais, ícones e mídias estruturais.
* `index.html`: Ponto de entrada da aplicação (página principal do ecossistema).
